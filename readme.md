## Overview 🎉
Typescript app, created to understand how much React makes life easier. I learned a lot of typescript/javascript too. 

## List of functions 📃
- Adding items to cart
- Filter
    - Star count
    - Min. price
    - Max. price

## Tech used 🔧
- Typescript
- SASS
- Font Awesome
